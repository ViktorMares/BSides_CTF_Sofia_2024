# Installation & Usage
```
git clone https://github.com/ViktorMares/BSides_CTF_Sofia_2024.git
```
```
cd BSides_CTF_Sofia_2024
```
```
pip3 install -r requirements.txt
```
```
cd app
```
```
uvicorn main:app --reload
```
